Woocommerce payment gateway for yTask. Connect your site to ytask and get payment flexibility. 

== Description ==
Woocommerce is a complete eCommerce solution for selling digital products on WordPress. Connect your site to ytask and get payment flexibility. yTask supports more than 20 payment methods you can shift between at will, while you enjoy the power of Woocommerce. Get all the financial and sales reports, analysis and engage clients like a pro through ytask.

yTask provides a complete payment system that enables you track all the financial information and sales that come from woocommerce.

== Installation ==
1. Upload the `ytask-payment-gateway-for-woocommerce` directory to  `/wp-content/plugins/`
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Create account with https://yTask.org and get your API key and  secret
4. Save the API key and secret in your woocommerce payment settings and hurray!
